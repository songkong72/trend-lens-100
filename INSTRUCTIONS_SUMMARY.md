# Trend-Lens 100 프로젝트 지시 사항 정리

이 문서는 지금까지 진행된 "Trend-Lens 100" 유튜브 인기 영상 통계 서비스의 개발 지시 사항 및 구현 현황을 정리한 것입니다.

## 1. 프로젝트 초기 설정 및 기술 스택
- **프레임워크**: Next.js (App Router), TypeScript
- **스타일링**: Tailwind CSS (Vanilla CSS 기반 전용 디자인 시스템 구축)
- **주요 라이브러리**:
  - `lucide-react`: 아이콘 시스템
  - `framer-motion`: 프리미엄 인터랙션 및 애니메이션
  - `recharts`: 데이터 시각화 (준비 완료)
  - `next-themes`: 다크 모드 지원
- **디자인 아이덴티티**:
  - **테마**: 프리미엄 다크 모드 (Premium Dark Mode)
  - **포인트 컬러**: 유튜브 레드 (#FF0000 계열, oklch 정의)
  - **효과**: Glassmorphism (배경 블러 및 반투명 효과)

## 2. 레이아웃 및 디자인 시스템 지시 사항
- **글로벌 스타일링**:
  - `oklch` 기반의 색상 변수 설정 (배경, 카드, 브랜드 컬러 등).
  - 전체 웹사이트에 Glassmorphism 유틸리티(`glass`) 적용.
  - 프리미엄 감각의 커스텀 스크롤바 및 인터랙티브 애니메이션(shake 등) 추가.
- **루트 레이아웃**:
  - 한국어(`lang="ko"`) 설정 및 SEO 최적화 메타데이터 적용.
  - Inter 폰트 적용 및 PWA 설치 환경 구성.

## 3. 사이드바 및 메뉴 최적화
- **메뉴 항목 구성**:
  - `[홈]`, `[트렌드 리포트]`, `[시청자 분석]`, `[설정]` 필수 포함 (한국어 라벨).
- **아이콘 라이브러리**: `Lucide-react`를 사용하여 메뉴별 적절한 아이콘 배치 (`Home`, `TrendingUp`, `Users`, `Settings`).
- **인터랙션 요구사항**:
  - **Hover 효과**: 마우스 오버 시 배경색(`bg-white/5`) 및 스케일 변화.
  - **Active 상태**: 클릭 시 활성화된 메뉴를 **유튜브 레드 컬러**로 강조 및 인디케이터 바 표시.

## 4. 컴포넌트별 상세 지시 사항 (고도화됨)
### [상단 헤더 - Header]
- **검색바**: "영상 제목, 채널명, 키워드"를 모두 검색 가능하다는 힌트 텍스트(Placeholder) 적용.
- **실시간 표시**: 우측 상단 `KOREA (LIVE)` 버튼 옆에 **마지막 데이터 업데이트 시각**(`Last Update 20:xx:xx`) 표시.
- **액션**: 알림 보드(애니메이션 포함), 사용자 프로필 Tier 및 계정 관리 UI.

### [트렌드 리포트 - TrendsView]
- **탭 인터페이스**: `실시간 인기 Top 100`과 `급상승 검색어` 두 개의 서브 탭으로 구성.
- **데이터 테이블**: 순위, 영상 제목, 채널명, 조회수, 등락 등을 포함한 상세 리스트 뷰(Table format) 구현.
- **라이브 트래킹**: 실시간 활성 지표 및 라이브 인디케이터 적용.

### [시청자 분석 - InsightsView]
- **검색 및 프로필**: 대형 채널 검색바와 분석 대상 채널의 구독자/조회수 요약 카드 제공.
- **성별 분포 (Pie Chart)**: 
  - 남성은 블루 계열(**#3B82F6**), 여성은 핑크/레드 계열(**#FB7185**)을 사용하여 직관성 확보.
- **연령대별 분포 (Bar Chart)**: 
  - 모서리가 둥근 **Rounded Bar** 스타일 적용.
  - **인터랙티브 툴팁**: 막대 오버 시 데이터 값이 팝업되는 효과.
- **비교 지표 (Trend Indicator)**: '전월 대비 15% 상승' 등 증감 수치를 보여주는 텍스트 및 아이콘 요소 배치.
- **기기별 분석**: 모바일/PC/태블릿 이용 비중을 진행률 바(Progress Bar) 형태로 시각화.

### [설정 - SettingsView]
- 시스템 업데이트 주기, 알림 활성화, 지역 설정, 프리미엄 계정 보안 등 관리 UI.

## 5. 현재 진행 상태
- 주요 4개 메뉴의 UI 구조 및 한국어 최적화 완료.
- 헤더 및 시청자 분석 페이지의 세부적인 디자인 디테일(컬러, 툴팁, 지표 등) 반영 완료.
- 프리미엄 다크 테마 및 애니메이션 시스템 고도화 완료.
